## Quick Canva/Figma Instructions

If creating in Canva:
1. Start with "Instagram Post (Square)" template
2. Set background to #1a1a2e
3. Add truck image (Canva Pro has stock photos)
4. Overlay dark gradient for text readability
5. Add text: Inter font, white, bold
6. Add yellow (#ffc107) button/CTA element
